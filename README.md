# Dead-Balls
Dead Ball is a round based top down zombie shooter that runs in your browser, just open / run the Play Dead Balls.HTML and have fun

*Install* -
Download files make a folder call it what ever put all the file inside and run the "Play Dead Balls.HTML"

*Controls* -
W A S D for moving, where youre mouse is thats where you aim / look, 1 to witch guns, escape (esc) to pause game, R to reload guns, and f5 to restart

*To load a Map* -
find the map forlder where youre game is and load it in

*Note* -
if any zombies get stuck they should respawn in an area next to you but if they are still stuck you can open a new tap in your browser and go back to the game and they will move through the walls (this is a bug but it helps)
